# 📱 College Food Delivery App

A student‑run campus food delivery service built with React, Node.js, and MongoDB.

---

## 🚀 Features

- **Browse** campus‑vendor menus  
- **Add to Cart** with live updates  
- **Favorites** toggle on each food item  
- **Vendor Dashboard** for order management
